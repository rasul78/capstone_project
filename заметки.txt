Вариант 5. Сменить позиционирование с «корпоративный чат» на agentic workflow
Сейчас Sentinel AI выглядит как «RAG-чат для документов». Эксперт хочет «agentic flow». Можно переподать тот же проект как:
«Sentinel AI — Multi-Agent Corporate Knowledge Assistant» где:

Planner Agent — разбивает вопрос на подзадачи
DocSearch Agent (via custom MCP) — ищет в внутренних документах
Legal Agent (via custom MCP) — проверяет соответствие праву РК
WebResearch Agent — добирает актуальные данные
Synthesis Agent — собирает финальный ответ
Critic Agent — проверяет ответ на галлюцинации

Это тот же код, но описанный как agentic flow. Поменять схему в README и Architecture Blueprint.
